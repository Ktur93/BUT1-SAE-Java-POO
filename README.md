# SAÉ — 2048 en Java (POO + tests JUnit)

Projet réalisé en **BUT1 Informatique (IUT de Montreuil)** dans le cadre de la **SAÉ 1.01**.  
Objectif : développer le jeu **2048** en **Java** en appliquant la **programmation orientée objet** et une démarche de **tests (JUnit)**, avec une logique de jeu fiable (déplacements, fusions, conditions de fin).

## Objectifs du projet
- Concevoir une solution en **POO** (classes, responsabilités, organisation du code)
- Implémenter la **logique complète** du 2048 (mouvements + fusions)
- Mettre en place des **tests** pour valider les comportements attendus
- Gérer les cas limites et améliorer la robustesse via le débogage

## Fonctionnalités
- Grille de jeu **2048** (initialisation et affichage)
- Déplacements dans les 4 directions (**haut / bas / gauche / droite**)
- Règles de fusion (addition de tuiles identiques)
- Génération d’une nouvelle tuile après un coup valide
- Conditions de fin :
  - victoire **2048**
  - défaite (plus de mouvements possibles)

## Technologies
- **Java**
- **JUnit** (tests unitaires)
